# Instaclone

Instagram clone with Express, Prisma, React, React Native

## User Stories

- [x] Create account
- [x] Request Secret
- [x] Confirm Secret (Log in)
- [x] Like / Unlike Photo
- [x] Comment on a Photo
- [x] Search by User
- [x] Search by Post
- [] See User profile
- [] Follow / Unfollow user
- [] See full photo
- [] Edit my profile
- [] Upload photos
- [] Edit the photo(include delete)
- [] See the feed
